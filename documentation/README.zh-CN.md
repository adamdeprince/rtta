# RTTA 研究信号说明

本目录记录了 RTTA 中一组研究原型交易信号。它们是信念更强、复杂度也更高的“登月式”指标，而不是通用图表指标。这些信号把市场微观结构研究、在线状态跟踪和审慎的不确定性门控组合进 RTTA 的增量 API。

每篇说明包括：

- 信号所依据的论文或研究思路；
- 实际实现的算法；
- 输出字段；
- 预期的数据形态和交易时段处理方式；
- 实现有意偏离论文之处。

## 一般性提醒

这些指标仍是研究原型。它们会输出信号、分数、目标仓位比例和流动性诊断，但不负责订单执行、投资组合构建、风险限额、融券可得性检查，也不建立交易成本模型（内部很小的 `cost_buffer` 阈值除外）。应将其输出视为特征或策略组件，而不是可独立运行的交易系统。
